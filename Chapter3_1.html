Vector Space

What is vector space?

A vector space is a non-empty set V of objects called vectors, on which two operations are defined, called addition and scalar multiplication, subject to the following axioms:

  1. 1. u + v = v + u;
  2. 2. (u + v) + w = u + (v + w);
  3. 3. There is a vector 0 in V such that u + 0 = 0 + u = u;
  4. 4. For each vector u in V, there exists a vector -u in V, such that u + (-u) = -u + u =0;
  5. 5. c(u + v) = cu + cv;
  6. 6. (c + d)u = cu + du;
  7. 7. c(du) = (cd)u;
  8. 8. 1*u = u

Using only these axioms, we can prove the zero vector in axiom 4 and the negative vector in axiom 5 -u are unique.

Subspace

In many problems, a vector space is composed of an appropriate subset of a large vector space. In this case, the ten axioms of vector space need only satisfy the following three properties to be called the subspace H of vector space V.

  1. 1. The zero vector in V is a member of H;
  2. 2. H is closed to vector addition: for any vector u and v in H, vector u+v belongs to H;
  3. 3. H is closed for vector multiplication: for any vector u and the scalar c in H, cu belongs to H.
