Vector representation

1. Definition and properties

We often put several values together and analyze them as a whole, which gives a definition - vector, an ordered list of values. The usual way to write a vector is to order them and enclose in parentheses or square brackets. For example, the following examples are a 2-dimensional vector, a 3-dimensional vector, and a 4-dimensional vector

The length and direction determine a vector. We can understand it better in terms of coordinate systems. In the two-dimensional coordinate system, the arrow represents a vector. It starts at the origin and the end point is the corresponding components of the vector. Each vector corresponds to a unique pair of numbers, and a logarithm in the coordinate system also corresponds to a unique vector.

     

As long as the magnitude and direction of the vector are the same, the vector is regarded as equal. As shown in the following figure, if a vector is moved randomly under the two-dimensional plane, the trajectory left by it will all be the same vector:
