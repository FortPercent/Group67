Matrix Operation

1. Simple operation

a) Get matrix elements

You can refer to the elements of a matrix by subscripts (column and column indexes) or indexs of the elements of a matrix. The index of an element is the order in which it is arranged in memory. In MATLAB, one index corresponds to one subscript. Take m*n matrix A as an example, the index of matrix element A(i,j) is (j-1)*m+ i. The mutual transformation relation can also be obtained by sub2ind and ind2sub. The example of the use of function sub2ind above shows that the element with subscript (1,3) in the matrix is indexed 7 in the Matlab memory. The element with subcript (2,2) is indexed 5 and subcript (3,1) is indexed 3.

b) Matrix splitting

The colon expression is used to obtain the submatrix:
A(:,j) : taking all elements in the jth column of matrix A;
A(i,:) : taking all elements in the ith row of matrix A ;
A(i,j) : taking the element in the ith row and jth column of A.

2. Matrix arithmetic operation

MATLAB basic arithmetic operations are: + (plus), - (minus), * (multiply), / or \(divide), ^ (power), '(transpose). Operation is carried out in the sense of matrix, the arithmetic operation of a single data is only a special case.

a) Matrix addition and subtraction: There are two matrices A and B, then the matrix addition and subtraction operations can be realized by A+B and A-B. The operation rule is that if the dimensions of A and B matrices are the same, the addition and subtraction of the matrices can be performed. If the dimensions of A and B are different, MATLAB will give an error message that the dimensions of the two matrices do not match.

b) Matrix multiplication: If A is a matrix with dimension m*n and B with dimension n*p, then C=A*B is a matrix with dimension m*p

c) Matrix division : There are two matrix division operations: \ and /, respectively representing left division and right division. If the A matrix is nonsingular, A\B and B/A can be realized. A\B is equivalent to the inverse A multiply B on the left, which is inv(A)*B. B/A is equivalent to inverse A multiply B on the right, which is B*inv(A). For matrices, generally, A\B≠B/A.

d) Matrix power: x power of matrix A can be expressed as A^x, The premise condition is that A is a square matrix and x is a scalar

e) Matrix transpose: swap the rows and columns of a real matrix. As for the complex matrix, conjugate is applied in addition.
