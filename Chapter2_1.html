Matrix Equation

What is matrix ?

If A is an m by n matrix, that is, there are m rows and n columns, then the ith row and jth column element of A is represented by aij, and the matrix A can be represented as A= [aij].

Matrix multiplication

A is an m by n matrix, B is an n by p matrix, and the columns of B are b1,b2... Bp, then the product AB is m by p matrix, its each column Ab1,Ab2... Abp, that is, AB=A [b1, b2... bp] =[Ab1, Ab2... Abp].
E.g.

Each column of AB is A linear combination of the columns of A, weighted by the elements of the corresponding column of B. The number of columns of A has to be equal to the number of rows of B for A linear combination to be defined, the number of rows of AB has to be equal to the number of rows of A, and the number of columns has to be equal to the number of columns of B.

Properties of matrix multiplication

  1. 1.Associative property of multiplication: A(BC) =(AB)C;
  2. 2.Left distributive property of multiplication: A(B+C) = AB+AC;
  3. 3.Right distributive property of multiplication :(B+C) A =BA+CA;
  4. 4.Identity of matrix multiplication: ImA =A =AIm.
