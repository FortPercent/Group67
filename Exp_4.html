Linear dependence

1. Definition

Given a set of vectors {vi } i=1,2,...,k if we can find a set of scalars {ci } i=1,2,...,k (excluding c1=c2=...=0 ) such that


                              c1v1+c2v2+...+ckvk=0

then the set of vectors is called linearly dependent. If there is none, they are linear independent. Here are some examples that help you have a better understanding.

v1 and v2 are linearly independent.
In fact, in two-dimensional plane, if two vectors are
not collinear, they are linear independent.
