What is linear equation?

The equations look like 'a1*x1+a2*x2+... +an*xn = b', 5*x1 - 6*x2 = x1*x2 and x1 + 6*x1*x2 = 3 are not linear equations.

When a system is composed of one or more linear equations containing the same variables, itâ€™s called System of linear equations.

Solution of the system:

substitute a set of Numbers for x1, x2... xn, all equations are equal to each other on both sides of the equation, and there are three possible scenarios: no solution (incompatible), only solution (compatible), infinitely many solutions (compatible)

To find the solution of a system of equations consisting of two linear equations with two variables is equivalent to finding the intersection of two lines:

only one solution (compatible)
no solution (incompatible)
infinitely many solutions (compatible)

Elementary row operations

Firt multiplexing: replace a row with the sum of multiples of itself and another row. Then substitution, replace two rows. At last, multiplication: multiply all the elements of a row by a non-zero number.
